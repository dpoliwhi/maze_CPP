Implementation of SmartCalc v2.0.

General information

The program inplement an engineering version of calculator and have a simpe interface with input window and buttons (digits, operations, functions, etc.).
The engine of program works by Dijkstra's algorithm and every expresson translate into Reverse Polish (bracketless) notation and then calculate.
Also SmartCalc builds graphs in new window with coordinate grid if input string has unknown variable (x).

Operations

SmartCalc has ordinary operations with nambers, like *, /, +, -, mod, pow.
User can prioritize operations by using (, ). Also user can change sign of last input number by press button +/-.
For delate last symbol use C, and fo clearing all input string use AC.

Functions

SmartCalc has opportunity to calculate some trigonometric functions, like cos(x), sin(x), tan(x), acos(x), asin(x), atan(x), sqrt(x), ln(x), log(x).
User can get a result with any number or can input unknown variable (x) and SmartCalc creates the graph of this function.
